# FilaFácil

Sistema simples de gerenciamento de filas para pequenos estabelecimentos, permitindo organizar atendimentos de forma rápida e eficiente.

---

## 📌 Sobre o Projeto

O FilaFácil tem como objetivo resolver a desorganização em filas de atendimento, oferecendo uma solução web simples para:

- Gerar senhas
- Organizar a fila
- Chamar clientes

O foco do projeto não é apenas o produto, mas também a aplicação de boas práticas de **processo de desenvolvimento de software**.
